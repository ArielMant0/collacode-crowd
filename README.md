## Description

This project provides a web-based interface to collect similarity judgments.
It contains two different approaches for data collection and was used in conjunction with Prolific to crowdsource similarity judgments for video games.

## Installation

Download the code from this repository. Then you can run the system on your machine directly.

### Manual Setup

To setup the frontend, simply install the required node packages using the package manager of your choice.

```bash
# setup with npm
npm install

# setup with yarn
yarn install
```

## Usage

CollaCrowd uses the standard VITE server for the frontend.


Debug Configuration

```bash
# frontend dev server using npm
npm run dev

# frontend dev server using yarn
yarn dev
```

Production Configuration

```bash
# frontend dev server using npm
npm run build
npm run preview

# frontend dev server using yarn
yarn build
yarn preview
```
